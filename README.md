# O Investigador da Qualidade

Aplicativo mobile-first inspirado no Akinator, mas focado em descobrir a causa raiz de problemas reais de empresas e consumidores.

## O que ele faz

- Coleta o perfil do usuário e o problema inicial.
- Faz perguntas curtas com respostas simples.
- Classifica hipóteses por categorias de Qualidade.
- Entrega causa raiz provável, evidências, indicadores e plano 5W2H.
- Permite copiar ou baixar o diagnóstico.

## Como rodar

Abra `index.html` no navegador.

Para publicar no GitHub Pages, envie estes arquivos para um repositório e ative Pages usando a branch principal.
