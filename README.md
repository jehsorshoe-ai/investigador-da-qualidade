# O Investigador da Qualidade

Aplicativo mobile-first inspirado no Akinator, mas focado em descobrir a causa raiz de problemas reais de empresas e consumidores.

## O que ele faz

- Coleta o perfil do usuário e o problema inicial.
- Adapta a investigação por segmento, área e público afetado.
- Faz perguntas curtas com respostas simples.
- Classifica hipóteses por um Ishikawa adaptado ao contexto.
- Combina leitura de Qualidade com 5 Porquês para aprofundar causa raiz.
- Entrega causa raiz provável, evidências, indicadores e plano 5W2H.
- Permite copiar ou baixar o diagnóstico.

## Lentes atuais

- Vendas / comercial: funil, oferta, mercado, leads, CRM, rotina e proposta de valor.
- Atendimento: expectativa, tempo de resposta, promessa e experiencia.
- Operação: método, entrada, ferramenta, medição e rotina.
- Produto / serviço: falha, critérios, uso, fornecedor e percepção do cliente.
- Pessoas: treinamento, liderança, carga e variação de execução.
- Financeiro: medição, margem, preço, rotina e impacto.

## Como rodar

Abra `index.html` no navegador.

Para publicar no GitHub Pages, envie estes arquivos para um repositório e ative Pages usando a branch principal.
